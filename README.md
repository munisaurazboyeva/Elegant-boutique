# Boutique Management System 👗

Ushbu loyiha kiyim-kechak do'konlari (boutique) uchun mahsulotlar zaxirasini (inventory) va sotuvlarni boshqarishga mo'ljallangan avtomatlashtirilgan tizimdir.

## ✨ Xususiyatlari:
- **Mahsulotlar nazorati:** Yangi mahsulot qo'shish, o'chirish va tahrirlash.
- **Brend bo'yicha saralash:** Har bir kiyimni o'z brendi bilan saqlash.
- **Real-vaqt statistikasi:** Umumiy mahsulotlar soni, jami sotuvlar va tushumni avtomatik hisoblash.
- **Zaxira ogohlantirishi:** Omborda 3 tadan kam qolgan mahsulotlarni maxsus belgi bilan ko'rsatish.
- **Chegirmalar tizimi:** Mahsulotlarga chegirma foizlarini qo'llash va yakuniy narxni hisoblash.
- **Qidiruv tizimi:** Nomi, brendi yoki rangi bo'yicha tezkor qidiruv.

## 🛠 Texnologiyalar:
- **Frontend:** HTML5, CSS3 (Modern UI/UX), JavaScript (ES6+).
- **Backend/Desktop:** Python 3.x, Tkinter (GUI kutubxonasi).
- **Database:** SQLite (Desktop uchun) va LocalStorage (Web uchun).

## 📂 Loyiha tuzilishi:
- `index.html` - Web ilovaning asosiy sahifasi.
- `script.js` - Logika va ma'lumotlar bilan ishlash.
- `style.css` - Premium dizayn va animatsiyalar.
- `boutique/main.py` - Desktop ilovaning Python kodi.
- `boutique/database.py` - SQLite bazasi bilan ishlash moduli.
